# MyHeart - a socially-economically aware heart disease risk calculator

https://icexorb.github.io/myheart

## Requirements
- Python3.xx (3.10 is recommended)
- pip
- scikit-learn, pandas, Flask

## Usage
### Backend:
`cd flask`

`python3 front-end.py`

## Frontend:
`cd web/src`

`npm install`

`npm start`
